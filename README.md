A **React + Vite** car rental application that lets users browse vehicles, check availability, and reserve rental dates. The app features a **clean, responsive UI** and a smooth user experience.

**Live Demo:** [car-rental-mu-gilt.vercel.app](https://car-rental-mu-gilt.vercel.app)

## Features
- Browse available cars with images and details  
- Check availability for specific dates  
- Reserve and book cars online  
- Responsive design for mobile, tablet, and desktop  
- Clean UI built for ease of use

---

## Technologies
- **Frontend:** React, Vite  
- **Styling:** CSS  
- **Linting:** ESLint  
- **Build Tool:** Vite  
- **Deployment:** Vercel

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Ilyasisse/CarRental.git

   
Navigate to the project folder:

2. cd CarRental

Install dependencies:

3. npm install

Start the development server:

4. npm run dev

Usage

Once the development server is running, open your browser and navigate to http://localhost:5173 (or the port Vite provides) to view the app.
